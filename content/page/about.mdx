## What is Tina CMS?

<Blockquote>
  Tina CMS, much like a guiding star, is a headless content management
  system designed to simplify the process of managing content on
  static websites. Unlike traditional CMS platforms, Tina embraces the
  concept of a headless architecture, providing developers with the
  flexibility to use their preferred frontend framework while still
  enjoying the benefits of a robust content management system.
</Blockquote>

## Features that Illuminate Tina CMS:

### 1. Intuitive Editing Interface:

Tina CMS offers a user-friendly editing interface that empowers content creators to make changes directly within the live site, providing a WYSIWYG (What You See Is What You Get) experience.

```javascript
// Example of Tina CMS Inline Editing
import { InlineWysiwyg } from "react-tinacms-inline";

const MyComponent = () => {
  return <InlineWysiwyg name='content' format='markdown' />;
};
```

### 2. Real-time Previews:

Say goodbye to guesswork! Tina CMS allows you to preview changes instantly, ensuring that your content looks pixel-perfect before publishing.

```javascript
// Example of Tina CMS Real-time Previews
import { useCMS } from "tinacms";

const MyPreviewButton = () => {
  const cms = useCMS();

  const preview = () => {
    cms.enable();
    cms.sidebar.show();
  };

  return <button onClick={preview}>Preview Changes</button>;
};
```

### 3. Customizable Content Fields:

Tailor your content editing experience with custom fields that match your specific project requirements, giving you unparalleled control over your content.

### 4. Git Integration:

Tina CMS seamlessly integrates with version control systems like Git, enabling developers to track changes and collaborate effectively within their preferred workflow.

```javascript
// Example of Git Integration with Tina CMS
import { GitClient } from "@tinacms/git-client";

const gitClient = new GitClient({
  path: "/path/to/repository",
  token: "your-github-token",
});

gitClient.on("onCommit", (commit) => {
  console.log("New commit created:", commit);
});
```
